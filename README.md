# Castellan 🏰

**Autonomous Agent Runtime for the Digital Kingdom**

*The Castellan guards the castle while the King sleeps.*

---

## What Is This?

Castellan is a persistent, autonomous agent framework. Agents that:
- Run 24/7 without human prompting
- Remember everything across sessions
- Improve themselves over time
- Spawn specialists for specific domains
- Act with delegated authority

Not chatbots. **Digital stewards.**

---

## Origin

Inspired by [Conway Research's Automaton](https://github.com/Conway-Research/automaton), rebuilt from scratch with:
- Stronger safety architecture
- No vendor lock-in
- No crypto/survival economics theater
- Integration with OpenClaw ecosystem
